# Contact_Management_Program

# ✏️ 교육과제 1~3 차 과제 ✏️
### 연락처 관리 프로그램 구축

# 과제 개요
### 이 과제의 목적은 수업 시간에 배운 자바 언어와 HashMap 사용해서 동작원리를 이해하고 응용능력을 키우는 것입니다. 더 나아가, 데이터베이스와의 연동을 통해 안전하고 효율적인 데이터 관리 기술을 습득하고, 스프링 부트를 활용한 웹 애플리케이션 개발로 이를 종합적으로 실습합니다.
